# QmeraTool

Qmera is a toolset for building quick and reliable application in dotnet.

## Features

- Fast Create Scaffolding
- Auto FluentValidation with minimal API middleware
- FluentValidation on OpenAPI, allows rich API documentations and generate validadors over the OpenAPI.
- Problem Details enabled by default
- Exception Handler Middleware for unhandled exceptions
- Enums input and output as String
- OpenTelemetry by default

## Installation

Install Qmera template

```bash
dotnet new install Qmera.Template
```

## Usage

```bash
dotnet new qmera -n MyApp
```
